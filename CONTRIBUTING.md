# Contributing to Tetris Game

Thank you for your interest in contributing to this Tetris game! Here are some guidelines to help you get started.

## Getting Started

1. **Fork the repository** and clone it locally
2. **Install dependencies**: `npm install`
3. **Run the development server**: `npm run dev`
4. **Make your changes** and test them thoroughly

## Development Setup

```bash
# Clone the repository
git clone https://github.com/yourusername/tetris.git
cd tetris

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Code Style

- Use meaningful variable and function names
- Follow React best practices and hooks patterns
- Keep components focused and maintainable
- Comment complex logic, especially game mechanics
- Use consistent indentation (2 spaces)

## Commit Messages

Please use clear and descriptive commit messages:

- `feat: Add new feature`
- `fix: Fix bug description`
- `docs: Update documentation`
- `style: Format code`
- `refactor: Refactor code without changing functionality`
- `test: Add or update tests`
- `chore: Update build tasks, dependencies, etc.`

## Pull Request Process

1. **Create a feature branch** from `main`
2. **Make your changes** with clear commits
3. **Test your changes** thoroughly
4. **Update documentation** if needed
5. **Submit a pull request** with a clear description

## Testing

Before submitting a PR, please test:

- All game controls work correctly
- Piece rotation and wall kicks function properly
- Scoring and level progression work as expected
- UI remains responsive on different screen sizes
- No console errors or warnings

## Areas for Contribution

- **Bug fixes**: Check the issues page
- **Features**: Mobile touch controls, new game modes, etc.
- **Performance**: Optimize rendering or game logic
- **Documentation**: Improve README, add tutorials
- **Testing**: Add unit or integration tests
- **Accessibility**: Improve keyboard navigation, screen reader support

## Questions?

Feel free to open an issue for any questions or discussions!

## Code of Conduct

Please be respectful and constructive in all interactions. We're all here to build something fun together!
